# tt2ff

A tool to synchronize Twitter to Fanfou

## Usage

```shell
# Clone this repo
$ git clone git@github.com:LitoMore/tt2ff.git

# Install dependencies
$ cd tt2ff
$ npm i

# Modify the config file according to your needs
$ cp config.js.example config.js

# Happ hacking!
$ node app.js
```
